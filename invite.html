📱 기기별 설치 방법
1
Safari로 앱 주소 열기
아래 앱 열기 버튼을 클릭하거나, Safari 브라우저에서 직접 주소를 입력하세요.
⚠️ 크롬이 아닌 Safari를 사용해야 홈 화면에 추가할 수 있습니다.
2
초대코드 입력
앱이 열리면 초대코드 입력 화면이 나타납니다.
위에 표시된 코드를 입력하고 입장하기 를 누르세요.
3
홈 화면에 추가
Safari 하단 가운데 ⎙ 공유 버튼 클릭
→ 스크롤 내려서 홈 화면에 추가 선택
→ 추가 클릭
✅ 이제 홈 화면에 앱 아이콘이 생성됩니다!
4
홈 화면 아이콘으로 실행
다음부터는 Safari 없이 홈 화면의 앱 아이콘을 눌러서 바로 실행하세요.
1
Chrome으로 앱 주소 열기
아래 앱 열기 버튼을 클릭하거나, Chrome 브라우저에서 직접 주소를 입력하세요.
2
초대코드 입력
앱이 열리면 초대코드 입력 화면이 나타납니다.
위에 표시된 코드를 입력하고 입장하기 를 누르세요.
3
앱 설치 (홈 화면 추가)
Chrome 주소창 오른쪽 ⊕ 설치 아이콘 클릭
또는 우상단 메뉴 ⋮ → 앱 설치 선택
→ 설치 클릭
✅ 홈 화면 & 앱 서랍에 아이콘이 추가됩니다!
4
앱 아이콘으로 실행
다음부터는 홈 화면의 앱 아이콘을 눌러서 바로 실행하세요.
1
Chrome에서 앱 열기
아래 앱 열기 버튼을 클릭하거나, Chrome 주소창에 직접 입력하세요.
2
초대코드 입력
앱이 열리면 초대코드 입력 화면이 나타납니다.
위에 표시된 코드를 입력하고 입장하기 를 누르세요.
3
앱 설치 (바탕화면에 추가)
Chrome 주소창 오른쪽 끝 ⊕ 설치 아이콘 클릭
또는 주소창 우측 메뉴 ⋮ → 제이스튜디오 캘린더 설치
→ 설치 클릭
✅ 바탕화면 & 시작 메뉴에 앱이 설치됩니다!
4
앱 아이콘으로 실행
다음부터는 바탕화면의 앱 아이콘을 더블클릭해서 실행하세요.
브라우저 없이 독립 앱처럼 실행됩니다.
🚀 앱 열기 (코드 자동 입력)

버튼을 누르면 초대코드가 자동으로 입력됩니다.
이후 화면 안내에 따라 설치를 완료하세요.
